# ctf-traces

A small, story-driven CTF you play in your browser. You're following the trail of **KAMAL**, a paranoid security researcher who went quiet. Each flag is a piece of evidence you recover.

## Style

- Beginner-friendly OSINT / web challenges, getting harder over time.
- Flag format: `traces{...}`
- No special tooling required.

## Run it

```sh
pip install -r requirements.txt
python app.py
```
